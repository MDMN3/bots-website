Juzo

Juzo is a Discord Bot for counting, anonymous confessions, suggestions, reacting with custom emojis and more.



About

Juzo is a general purpose Discord bot. I started making this bot for fun and because I can. There are already a lot of moderation and music bots. That's why I wanted this bot to focus on fun features to make my server more colorful and entertaining. After implementing and adding more features I realized that some of the moderation features of other popular bots had some loopholes. Therefore I started implementing my own moderation tools.

Commands

This is my first bot. I'll put every command in that I like until it explodes. It's still in development. Many changes. Many updates. Chaos.

Command Description Example
botinfo Shows bot info j!botinfo
cat Shows a random pic of a cat j!cat
dog Shows a random pic of a dog j!dog
emojis Shows a list of all custom server emojis j!emojis
help Shows all commands j!help
ping Checks if Bot is online j!ping
react [emoji] Sends a reaction j!react gifcolorheart
reacttolast [emoji] Reacts to the last message with the given emoji j!reacttolast gifcolorheart
reacttothis [emoji] Reacts to the message with the given emoji j!reacttothis gifcolorheart
say [message] Lets the bot say something j!say hello
serverinfo Shows server info j!serverinfo
userinfo [@user] Shows user info j!userinfo
j!userinfo Peter
ytsearch [youtube-title] Searches for videos on YouTube j!ytsearch blackpink
